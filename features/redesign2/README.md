# Redesign 2 - Planning Documents

## Overview

This is the second redesign iteration for the videographer portfolio website. This redesign introduces a new color scheme with an off-white background (#C9C8C7) and restructures the navigation to include a Work page with sub-pages for Videos and Photos, plus a new "Other" page with a striking glitchy hero.

## Design Philosophy

Following the Carson-Oliver-West-Weirdcore methodology:
- **Carson**: Experimental typography, break conventions, asymmetrical layouts
- **Oliver**: Dark beauty, distressed textures, gothic elegance
- **West**: Minimalist perfectionism, bold statements, clean aesthetics
- **Weirdcore**: Glitch as art, intentional digital artifacts

## Key Changes

### Color Scheme
- **Background**: Off-white (#C9C8C7) - major shift from black
- **Text**: Dark text on light background (inverted contrast)
- **Accents**: Maintain red/green/pink for gore core, brat green, indie sleaze

### Page Structure
1. **Home** - Landing page with video grid
2. **Work** - Hub page with navigation to Videos/Photos
   - **Videos** - Video portfolio page
   - **Photos** - Photo portfolio page (dark background with grain)
3. **Other** - New page with "LOST FILES" glitchy hero (dark background with grain)
4. **Contact** - Contact form with dark aesthetic and grain texture (matching Photos and Lost Files)
   - FormSubmit integration with CAPTCHA protection
   - Email: `imanolV20@icloud.com`
   - Thank you page with navigation options
